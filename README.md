# ---reaProgramador
Esse projeto foi feito no Visual Studio 2022 com o Windows Form.
Pode preencher o nome, qual linguagem usar, quanto tempo de experiência, e qual cargo entrar.
Quando preencher e clicar no botão, vai falar sobre o usuário, sobre a linguagem escolhida e a média salarial de acordo com o cargo,
que pode variar de localidade, linguagem escolhida e a área que está trabalhando.
Sobre a as descrições das linguagens mencionadas, a pessoa pode ter uma opinião diferente sobre campo descrito que fala sobre a linguagem escolhida.
